# pardos-chicken

Proyecto backend + frontend para clientes y restaurante.

## Frontend clientes

Este módulo usa **React + Vite** como base.  
Incluye configuración mínima para trabajar con HMR y ESLint.  

Actualmente, dos plugins oficiales están disponibles:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react)  
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc)  

## React Compiler

El compilador de React no está habilitado en esta plantilla por temas de rendimiento.  
Para activarlo, revisa la [documentación oficial](https://react.dev/learn/react-compiler/installation).
